
# Pariposhan | Building Safer Food Culture

Pariposhan is a community-driven platform designed to foster a safer food culture through shared insights, verified news, and educational tools.

## Key Features
- **Community Feed**: Real-time food safety alerts and nutrition tips shared by members.
- **Knowledge Hub**: Curated deep-dives into health, ecology, and science.
- **GoodKart**: A community-audited directory of sustainable and verified food products.
- **AI Safety Assistant**: Integrated Hugging Face chatbot for instant safety queries.
- **Inspirational Header**: Featuring daily wisdom on intelligent eating.
- **Quiz & Arcade**: Learn food safety through weekly challenges and retro mini-games.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Database/Auth**: Firebase Firestore & Firebase Auth
- **Styling**: Tailwind CSS & ShadCN UI
- **AI**: Genkit (News Fetching) & Hugging Face (Chatbot)

## Local Development
1. **Extract**: Unzip the project folder.
2. **Install**: Run `npm install` in your terminal.
3. **Run**: Run `npm run dev` to start the development server.
4. **View**: Open [http://localhost:9002](http://localhost:9002).

## Deployment Guide (Firebase App Hosting)
1. **GitHub**: Push your project to a new GitHub repository.
2. **Firebase Console**: Navigate to your project and select **App Hosting**.
3. **Connect**: Link your GitHub repository.
4. **Auto-Deploy**: Firebase will automatically detect the Next.js settings and deploy your application.
